Endpoints disponibles:

GET /health - Verificar el estado de la aplicación

Ejemplo: {{ url_for('web.health_check') }}

GET /api/medios - Listar todos los medios registrados

Ejemplo: {{ url_for('api.listar_medios') }}

GET /api/temas - Obtener todos los temas

Ejemplo: {{ url_for('api.obtener_temas_flexibles') }}

GET /api/temas?tipo=propio - Filtrar temas por tipo de medio

Ejemplo: {{ url_for('api.obtener_temas') }}?tipo=propio

GET /api/temas?dominio=informacion.es - Obtener temas por dominio (uso con extensión)

Ejemplo: {{ url_for('api.obtener_temas_por_dominio') }}?dominio=informacion.es

GET /visualizar - Ver una página con los temas monitoreados

Ejemplo: {{ url_for('web.visualizar_temas') }}
